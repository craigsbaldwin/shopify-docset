discount

This page was printed on Jan 23, 2020. For the current version, visit https://help.shopify.com/en/themes/liquid/objects/discount.

The discount object contains information about a discount, including its id, code, amount, savings, and type. Note that this object will display a value only if it's accessed in notifications or in the Order Printer app.

Note

Attributes for money amounts, such as prices and totals, are in the customer's local (presentment) currency. Use a money filter to show the result as a monetary amount.

The discount object has the following attributes:

On this page

  • discount.amount
  • discount.code
  • discount.id
  • discount.savings
  • discount.title
  • discount.total_amount
  • discount.total_savings
  • discount.type

discount.amount

Returns the amount of the discount.

Input

Output

discount.code

Returns the title or discount code of the discount. Same as discount.title.

Note

In email notifications templates, discount.code must be in a loop to appear correctly. This allows for multiple discounts to be applied.

discount.id

Returns the id of the discount.

discount.savings

Returns the amount of the discount's savings. The negative opposite of amount.

Input

Output

discount.title

Returns the title or discount code of the discount.

Input

Output

discount.total_amount

Returns the total amount of the discount if it has been applied to multiple line items.

discount.total_savings

Returns the total amount of the discount's savings if it has been applied to multiple line items. The negative opposite of total_amount.

discount.type

Returns the type of the discount. The possible values of discount.type are:

  • FixedAmountDiscount
  • PercentageDiscount
  • ShippingDiscount
