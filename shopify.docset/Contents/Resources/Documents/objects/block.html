block

This page was printed on Jan 23, 2020. For the current version, visit https://help.shopify.com/en/themes/liquid/objects/block.

A block represents the content and settings of a single block in an array of section blocks. The block object can be accessed in a section file by looping through section.blocks.

The block object has the following attributes:

On this page

  • block.id
  • block.settings
  • block.shopify_attributes
  • block.type

block.id

Returns a unique ID dynamically generated by Shopify.

block.settings

Returns an object of the block settings set in the theme editor. Retrieve setting values by referencing the setting's unique id.

Input

Output

block.shopify_attributes

Returns a string representing the block's attributes.

The theme editor's JavaScript API uses a block's shopify_attributes to identify blocks and listen for events. No value for block.shopify_attributes is returned outside the theme editor.

block.type

Returns the type defined in the block's schema. This is useful for displaying different markup based on the block.type.
